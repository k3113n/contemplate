# contemplate
A template node.js server with a Vite + React frontend, built with concurrently


## usage
- initial setup
```bash
git clone https://github.com/k3113n/contemplate
cd contemplate
npm link
```

- create app from template
```bash
npx create-contemplate your-app-name
```

- run your new app
```bash
cd your-app-name
npm start
```
